# Virtual Classroom Manager (Microsoft Teams App)

## Problem Statement

## Solution Description

- Interactive whiteboards
- Breakout rooms
- Attendance tracking
- Assignment management
- Push notifications
- Offline mode
- Touch-optimized UI
